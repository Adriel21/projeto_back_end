
# Notações sobre o Projeto de Back-End - Senac

## Sobre a modelagem e organização do banco
- O banco de dados, para este projeto, não se encontra normalizado. A organização fora pensada para três tabelas, sendo essas: usuario, projeto e categoria. 

---
## Sobre o acesso das informações do banco
- A conexão com o banco e o acesso as suas informações serão feitas através de classes. As classes pensadas para que ocorra tal procedimento são: Banco - onde será programado a conexão com o banco. Usuario, para acesso as informações da tabela usuário. Categoria e Projeto.
